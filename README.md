# Signals
Project for Signals, Image &amp; Video, Master's Degree in Artificial Intelligence Systems

## Setup
Clone repository
```
git clone git@github.com:Nathanoj02/signals.git
```

Update submodule
```
git submodule update --init --recursive
```

Cpp setup
```
mkdir build && cd build
```
```
cmake ..
```
```
make -j
```
